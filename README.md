# Odin Recipes

This is my first project in The Odin Project Foundations course.

In this project, I will build a basic recipe website using HTML.

The website will consist of a main index page with links to several recipes.

By completing this project, I will demonstrate basic HTML structure and the use of links.

I expect to learn how to use basic HTML.

The challenge will be to use as little help as possible and ask myself the right questions to find the right solutions.

This is important to me because I want to learn and develop myself.

## Reflection


After completing this project, I learned:

How to apply everything that was covered in the lessons into one complete project. This was the moment when everything really started to make sense for me.

How an index.html file and separate recipe pages are connected through links and folder structure.

How to properly use headings, paragraphs, images (including the alt attribute), and both unordered and ordered lists.

How to use Git more confidently, including staging changes, writing meaningful commit messages, and managing my workflow.

How to push my work to GitHub and publish my project online using GitHub Pages.

The importance of clean structure and organizing files correctly.

By repeating the concepts and practicing step by step, I started to understand them more naturally. At one point, I had a real “aha” moment where it all clicked.

Most importantly, I really enjoyed building this project. It made learning HTML, Git, and GitHub feel practical and motivating.
